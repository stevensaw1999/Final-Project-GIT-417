HTML and CSS Examples to Help You Get Started on the Final Project

These are just meant to help you when you begin the final project for GIT417. There is no JavaScript written, but if you struggle with HTML and CSS, these code examples are a start for the sections of the page you could use to build the product switcher, guessing game, or shopping cart. Be sure to update the code with appropriate content, ids and classes, and to write functional JavaScript to work with it for the final.Make sure that you understand how the styles are wokring and how the cascade is working here, as well as how CSS Reset works, because it was used in this pen, along with CSS variables and rem units. The code on this page is not responsive!
